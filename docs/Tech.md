**TLDR:**
This document specifies the technology stack used in the `GfX-Engine` project:
* Key technologies: Linux, SDL3, Rust

- Programming Language: Rust 1.89.0+ (Edition 2024)

- Graphics & Windowing: SDL3

- Audio Engine: Kira

- Audio Analysis: Spectrum-Analyzer (FFT), RustFFT, Hound

- Typography: RustType (TTF rendering)

- Data: Serde, Toml, Quick-XML

- Screen Canvas: Virtual 1920x1080 (HD), with 1:1 pixel mapping logic.
